# Crypto Tracker

A cryptocurrency tracking app built with Nuxt.js and CoinGecko API.

## Features

- Browse 1000+ cryptocurrencies sorted by market cap
- Search coins by name or symbol
- Infinite scroll pagination
- Responsive UI
- Detailed coin pages with:
  - Current price and 24h changes
  - 7-day price chart
  - Market cap, volume, and supply stats
- Dark theme UI
- Fully responsive

## Tech Stack

- Nuxt.js 2
- Tailwind CSS
- Chart.js

## Setup

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
